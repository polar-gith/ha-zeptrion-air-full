# Zeptrion Air Full

Custom Home Assistant integration for Feller Zeptrion Air.

## Features
- Covers / blinds
- Lights including dimming
- Zeroconf discovery
- Local communication

## Installation
1. Open HACS
2. Add this repository as a custom repository
3. Select category: Integration
4. Download and restart Home Assistant
